# Contributing

This is my personal Claude Code configuration — published as a reference, not a product. You're welcome to:

- **Fork it.** MIT-licensed; lift any rule, agent, hook, or playbook into your own `~/.claude`.
- **Open an issue** for a factual error, a broken link, or a doctrine doc that has drifted.
- **Send a PR** against the doctrine docs (`rules/`, `best-practice/`, `playbooks/`) or agent/command definitions. Run `scripts/audit-config.sh` before pushing — the config-drift gate must pass.

What I won't merge: changes to my personal identity/profile scaffolding, or features that only make sense for my workflow. Fork those instead.

No CLA, no formalities.
